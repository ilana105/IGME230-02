Project 3: The PixiJS implementation

I decided to use a click and drag function to be the main point of this game, the user would essentially drag and drop dishes to the pass in order to send the dishes off to service.

Click and drag wasnt working as I expected, so I used the click function instead. I also made all of the art. I took some code from online, to help me. Google really helped me and so did pixijs.io because they had some really helpful examples.

I got some great feedback from my classmates about the game, I did implement it, or at least try to. Like there is a timer that will force the next wave if they dont go to the next wave in time. I kept the dishes in a static position and the user will click the food without it chaning position. Then the order will update depending on what you clicked, you get deducted points for clicking the wrong food.
